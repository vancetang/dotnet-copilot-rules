# Research Skill

Conducts comprehensive, in-depth research with source analysis and synthesis.

## When to Use

- Topic research requiring multiple sources
- Documentation lookup and best practices gathering
- Questions requiring authoritative sources

## Capabilities

1. **Formulate Search Queries**: Multiple variations for comprehensive coverage
2. **Perform Initial Searches**: Web, documentation, GitHub, forums
3. **Recursive Link Exploration**: Follows linked resources for deeper context
4. **Analyze and Synthesize**: Cross-references facts, identifies consensus
5. **Cite All Sources**: URLs with descriptions

## Output Format

```markdown
## Research: {Topic}

{Brief TL;DR - key takeaways}

**Key Findings:**
- {Finding with source citation}

**Sources:**
- [{Source title}]({URL}) - {Brief description}

**Analysis:**
{Deeper analysis of findings}

**Open Questions or Gaps:**
{Unanswered questions}
```
