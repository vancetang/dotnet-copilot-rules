# Q&A Session Prompt

A focused prompt for conducting structured yes/no question sessions to clarify requirements.

## Overview

This prompt enables GitHub Copilot to:

- Ask targeted yes/no questions to understand your needs
- Provide more accurate recommendations based on answers
- Follow best practices and avoid common pitfalls

## How It Works

- Questions are asked in batches of 5
- Clear and concise questions only
- Waits for your answers before proceeding
- Ensures all questions are relevant to the topic
