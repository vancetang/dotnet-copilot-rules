# Commit Prompt

A prompt for creating well-formatted git commits with conventional commit messages and emojis.

## Overview

Use this prompt to:

- Create atomic, focused commits
- Follow conventional commit format
- Include descriptive emojis by commit type
- Generate meaningful commit messages

## Usage

```
/commit
```

Or skip pre-commit checks:

```
/commit --no-verify
```
