Learn how to customize GitHub Copilot for your projects.

GitHub Copilot can be customized to better understand your codebase and coding preferences. This section covers the basics of setting up custom instructions and configurations.

## Prerequisites

- GitHub Copilot subscription
- VS Code with GitHub Copilot extension installed
